# Every Kind of Volcano

## Overview
SciShow Kids 探索地球上各种类型的火山——从锥形火山到盾状火山，从活跃火山到休眠火山，甚至海底火山。通过 Jessi 和 Sam the Bat 的对话，孩子们了解火山的分类、喷发威力指数（VEI），以及火山如何塑造地球表面。

## Learning Value
- **听力**：学习火山类型（stratovolcano, shield volcano, fissure volcano）的专业英语词汇
- **口语**：模仿主持人对话中的幽默互动（Sam the Bat 的搞笑反应）
- **口译**：火山分类体系的中英转换练习（适合 L3 进阶）
- **科学素养**：理解 VEI 火山爆发指数、活火山/休眠火山/死火山的区别

## Difficulty Analysis
- **词汇**：中等偏高 — 包含 stratovolcano, shield volcano, caldera, fissure, magma, lava, dormant, extinct, VEI 等专业术语
- **句式**：对话体，有重复和解释，适合跟读
- **文化背景**：引用真实火山案例（富士山、夏威夷冒纳罗亚、庞贝古城、冰岛 Eyjafjallajökull、印尼坦博拉）
- **知识密度**：高 — 涵盖火山分类、喷发类型、历史记录、监测方法、火山益处

## Recommended Age
**8-10 岁（L2）** — 视频较长（8分钟），包含较多学科术语和类比（如 VEI 0-8 的分级），适合有一定英语基础的儿童。语速适中但信息量大，8岁以下可能需要家长陪同讲解。

## Usage Tips
- **Pre-listening**：先展示地球剖面图，介绍 magma/lava 的区别
- **During**：让孩子标记听到的每种火山类型，配合 [[fold-mountains]] 概念
- **Post-listening**：
  - L1 输出：跟读 "Some volcanoes erupt suddenly and some erupt slowly."
  - L2 输出：用自己的话复述三种火山类型的区别
  - L3 口译：模拟"小小地质学家"，将火山分类介绍从英文翻译成中文
